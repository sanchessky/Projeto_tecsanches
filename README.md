## Projeto_tecsanches
# Descrição
Projeto integrador do Senac Tatuapé, desenvolvido como Front-End do site Tecnologia do Sanches.
Este projeto tem como objetivo criar uma interface moderna, funcional e responsiva para uma loja online de tecnologia, combinando design atraente e usabilidade.

# Tecnologias Utilizadas
* HTML

* CSS3 (Flexbox, Grid, Responsividade)

* Banco de Dados: MySQL

# Funcionalidades Principais
* Layout inspirado em lojas de tecnologia modernas

* Navegação intuitiva e responsiva para dispositivos móveis

* Seções de produtos, categorias e destaques

* Rodapé com informações institucionais, formas de pagamento, segurança, suporte e redes sociais

* Ícones e elementos visuais personalizados

# Como Executar o Projeto
1. Clone este repositório:

`bash`
```
git clone https://github.com/Sanchessky/Projeto_tecsanches.git
```
2. Configure o banco MySQL conforme o arquivo de script (se houver).

3. Abra o arquivo index.html no seu navegador favorito para visualizar o Front-End.
